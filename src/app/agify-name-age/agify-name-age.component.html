Get a prediction of a person's age, gender, and nationality.

The webapp uses agify.io's free and public api.
You can also enable additional predictions by using checkboxes. Gender peridiction is provided by: genderize.io and nationality prediction by nationalize.io

  • The age for {{ name }} is: {{ age }}
  • {{ name }} is a {{ gender }} name.
  • There is a high chance that you are from {{ nationality }}.
